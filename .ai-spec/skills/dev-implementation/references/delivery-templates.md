# 实施交付模板

## 实施计划

```markdown
# 实施计划：{任务}

## 目标、范围与不改事项
## 现状证据与接入点
## 修改位置及依赖顺序
## 数据、契约、权限和兼容影响
## 测试与验证方法
## 风险、停止条件与回滚
```

## 缺陷修复说明

```markdown
# 缺陷修复：{问题}

## 现象与复现条件
## 根因证据
## 最小修复
## 修改文件
## 已执行验证与结果
## 相邻风险、未验证项与回滚
```

## 最终交付

```markdown
# 任务交付：{任务}

## 目标与实际完成范围
## 修改文件和关键行为
## 数据、契约、权限、安全与兼容影响
## 已执行命令和观察结果
## 未验证内容及原因
## 风险和已知限制
## 回滚步骤
## 剩余事项与责任人
```
