# 实施完成检查表

## 范围

- 修改只覆盖已批准目标，没有无关重构或批量格式化。
- 用户已有修改得到保留，生成文件和第三方代码未被误改。
- 事实、假设和未决事项均已记录。

## 一致性

- 实现遵循项目原生命名、分层、错误处理和命令。
- 代码、测试、契约、数据、权限和文档保持同步。
- 没有为通过检查而移除校验、授权、异常处理或审计。

## 验证

- 缺陷有复现与回归证据，新行为有可观察测试。
- 已执行命令和真实结果被记录，未执行内容没有伪装为通过。
- 正常、失败、边界、重复、并发和恢复路径按风险得到覆盖。

## 交付

- 影响范围、兼容性、风险、未验证项和责任人明确。
- 回滚步骤具体，数据恢复与代码回退已区分。
- 未经授权没有生产、破坏性、外部消息、付费或不可逆操作。
